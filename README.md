## 테스트 및 평가
- 프로젝트의 목표가 실시간 CCTV 탐지
   - CCTV의 특성
      - 실시간
      - 비교적 저화질
- 위같은 특성에 따라 추론속도가 빠르며, 저화질에서도 탐지성능이 준수해야함
- 이에 따라 가장 적절한 모델로 yolo에서 제공하는 `rtdtr-l.pt` 모델로 선정

## 프로젝트 소개

- 프로젝트 주제 : **인파밀집 경고 시스템**
- 프로젝트 명 : **관심영역(ROI) 기반 위험 예측 관제 시스템 개발**
- 프로잭트 목표 :
   - YOLO v11 모델을 **전이학습**시켜 **광장**, **골목길** 에서의 **이상징후를 탐지**하고, **이를 알리는 관제시스템**
   - **골목길**에서의 **인파밀집 예측 및 탐지**
   - **광장** 에서의 **이상 징후 탐지**
   - 해당 사항 탐지 시 **관제시스템에 알림 표출**
   - 전광판 및 알림판에 **탐지내용 송출**
